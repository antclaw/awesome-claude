## Getting Started

Claude Code is Anthropic's AI coding assistant that helps you write, review, and debug code. This repository collects the best skills, prompts, and resources for getting the most out of Claude Code.

### Prerequisites

- Claude Code installed
- Basic programming knowledge
- Git and terminal familiarity

### Installation

1. Clone this repository:
```bash
git clone https://github.com/antclaw/awesome-claude.git
cd awesome-claude
```

2. Explore the skills and prompts in the `skills/` directory

3. Start using Claude Code with the provided prompts
